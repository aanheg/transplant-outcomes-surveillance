# Data

Place these three files in this folder before running the notebook:

| File | Where to get it |
|---|---|
| `Kidney_Figures_Supporting_Information.xlsx` | srtr.hrsa.gov/adr → Kidney chapter → Excel download icon next to any figure |
| `Heart_Figures_Supporting_Information.xlsx` | srtr.hrsa.gov/adr → Heart chapter → Excel download icon next to any figure |
| `Liver_Figures_Supporting_Information.xlsx` | srtr.hrsa.gov/adr → Liver chapter → Excel download icon next to any figure |

No account, DUA, or IRB required. These are publicly posted aggregate data files.

The `.gitignore` excludes them from version control since they're freely available from the source.
