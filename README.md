## What this does

Builds a surveillance pipeline on publicly available SRTR/OPTN data to track post-transplant outcomes and detect deviations early — before they surface in published Program-Specific Reports.

Three detection methods run in parallel:

- **Statistical Process Control (SPC)** — rolling 3-year baseline with ±2 SD control limits on 6-month and 1-year post-transplant death rates. Standard methodology used in hospital QAPI programs.
- **Short-term vs long-term divergence** — flags years where 6-month mortality spikes disproportionately relative to 1-year survival. This is the signature of an acute procedural complication: something going wrong in surgery kills patients before 6 months, while those who survive past that tend to reach 1 year. A surgical tool causing 90-day complications would appear in this signal before propagating to long-term survival curves.
- **Delayed Graft Function (DGF) surveillance** — kidney-specific control chart for 90-day acute complications. DGF is defined as dialysis required within the first 7 days post-transplant and is the most direct measurable proxy for early post-operative complications.

### Key finding

Kidney DGF rates increased from **17.7% (2013) to 26.7% (2024)** — a **+9 percentage point / +51% relative increase** — with a sustained elevation beginning in 2020. This pattern, rising acute short-term complications without proportionate worsening of long-term survival, is consistent with a procedural or equipment factor affecting early post-transplant function and is detectable before long-term survival curves move.

| File | Description |
|---|---|
| `death_rates_clean.csv` | Post-transplant death rates at 6-month, 1-year, 3-year, 5-year by organ and cohort year |
| `tx_counts_clean.csv` | Annual transplant volumes by organ, 2013–2024 |
| `waitlist_mortality_clean.csv` | Waitlist death rates per 100 patient-years |
| `dgf_rates_clean.csv` | Kidney DGF rates 2013–2024 |
| `trend_analysis.csv` | SPC results with rolling mean and ±2 SD control limits |
| `divergence_analysis.csv` | Short-term vs long-term mortality ratio by organ and year |
| `dgf_trend_analysis.csv` | DGF control chart with severity flags |
| `flagged_signals.csv` | All anomaly detections ranked by severity (RED / YELLOW / GREEN) |
| `data_quality_report.json` | Audit trail of every missing value and cleaning decision |

## Data notes

- All source data is publicly available from SRTR/HRSA. No application, IRB, or DUA required.
- Missing values for recent cohort years (e.g. no 3-year outcomes for 2022 transplants) are expected and documented in `data_quality_report.json`.
- This analysis uses national aggregate data, not center-level data. Results describe national trends. Center-specific surveillance requires SRTR Standard Analysis File access.
